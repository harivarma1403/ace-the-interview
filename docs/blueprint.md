# **App Name**: Ace the Interview

## Core Features:

- Interview Simulation: Simulate interview questions based on the specified job description. Use the prompt to tailor the questions to the most important criteria for a particular role.
- Performance Evaluation: Record the user's interview performance, then use generative AI as a tool to evaluate it and create detailed feedback in a downloadable PDF report, containing assessments of specific qualities.
- Intuitive User Interface: Provide a clean and intuitive user interface for easy navigation and a professional feel, which avoids unnecessary complexity to deliver the feedback clearly.
- Downloadable Feedback Reports: Enable users to download their interview feedback as a PDF document for offline review and sharing. Focus on text-based feedback only in the MVP, without any inline styling or complex formatting.

## Style Guidelines:

- Primary color: Strong blue (#3498db) to convey professionalism and trustworthiness.
- Background color: Light gray (#f2f3f4) for a clean and unobtrusive backdrop.
- Accent color: Soft green (#2ecc71) for positive feedback highlights.
- Body and headline font: 'Inter' (sans-serif) for readability and a modern, machined look.
- Use clear and professional icons to represent different evaluation aspects.
- Implement a simple and well-structured layout with clear sections for questions, answers, and feedback.
- Subtle animations when transitioning between questions or showing feedback.
